SQL DAL Maker is a generator of DTO and DAO classes. Target programming languages: PHP, Java, C++, Python, Ruby.

To generate the class, you declare it in XML meta-program.

Project Website: http://sqldalmaker.sourceforge.net

SQL DAL Maker for Eclipse: http://marketplace.eclipse.org/content/sql-dal-maker
SQL DAL Maker for IntelliJ-Platform: https://plugins.jetbrains.com/plugin/7092
